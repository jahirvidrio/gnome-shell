# Gnome Shell | Themes


## Install themes

```shell
git clone git@github.com:jahirvidrio/gnome-shell.git ~/.my-gnome-shell
ln -s ~/.gnome-shell-themes/icons ~/.icons
ln -s ~/.gnome-shell-themes/themes ~/.themes
```

## Install extensions

- Big Sur Status Area
